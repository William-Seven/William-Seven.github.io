要证明这个定理，我们可以通过数学归纳法来进行证明。首先，我们可以考虑递推方程：
$[a_n = c_1 a{n-1} + c 2 a_{n-2}]$
我们已知特征方程 (r^2 - c 1 r - c 2 = 0) 有一个特征根 (r 0)。我们要证明，如果这个特征方程只有一个特征根 (r 0)，那么递推方程的通解可以表示为：
$a_n = \alpha_1 r_0^n + \alpha_2 n r_0^n$
其中，(\alpha 1) 和 (\alpha 2) 是常数。
基本步骤：

  1. 初始情况： 首先，我们验证当 (n = 0) 和 (n = 1) 时，递推方程的通解是否成立。这是数学归纳法的基础。
  2. 归纳假设： 假设对于某个正整数 (k)，递推方程的通解成立，即：
    $a_k = \alpha_1 r_0^k + \alpha_2 k r_0^k$
  3. 归纳步骤： 我们考虑 (n = k + 1) 的情况，即：
    $a_{k+1} = c_1 a_k + c_2 a_{k-1}$
    使用归纳假设，将 (ak) 和 (a{k-1}) 替换为它们的表达式，然后化简，最终得到类似于 (a{k+1} = \alpha 1 r 0^{k+1} + \alpha 2 (k+1) r_0^{k+1}) 的形式。
    这样，通过数学归纳法，我们可以证明递推方程的通解。
    请注意，这个证明的详细步骤可能会因为具体的递推方程形式而有所不同，但基本思路是相似的。你可能需要使用一些初等代数和数学归纳法的技巧来完成证明。
